=====
round
=====

`round` clusters the placements by rounding branch lengths.

::

  usage: round [options] placefile[s]

Options
=======

-o  Set the prefix to write to. Required.
--sig-figs  Set the number of significant figures used for rounding (default 3).
--cutoff  Set the rounding inclusion cutoff for the ML weight ration (default 0.01).
